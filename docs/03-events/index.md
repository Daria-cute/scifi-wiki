# ⏳ 科幻文学历史对照大事记

下表按照时间顺序，对比列出了世界科幻与中国科幻发展中的关键节点，呈现了两者从“错位”到“融合”的历史进程。

| 时间 | 🌍 世界科幻大事件 | 🇨🇳 中国科幻大事件 |
| :--- | :--- | :--- |
| **1818年** | **科幻元年**<br/>玛丽·雪莱**《弗兰肯斯坦》**出版，被公认为世界第一部科幻小说。 | — |
| **1865/<br/>1903年** | 儒勒·凡尔纳**《从地球到月球》**发表。 | **译介开端**<br/>鲁迅根据日文版翻译凡尔纳的《从地球到月球》，译名**《月界旅行》**。他在序言中提出科幻小说能“导中国人群以进行”，这是中国最早讨论科幻文学的文字。 |
| **1904年** | — | **原创萌芽**<br/>荒江钓叟在《绣像小说》发表**《月球殖民地小说》**，被普遍认为是中国本土原创科幻的开端。 |
| **1926年** | **类型定名**<br/>美国出版人雨果·根斯巴克创办杂志**《惊奇故事》**，首次提出并推广“Science Fiction”这一名称。 | — |
| **1932年** | — | **文学化探索**<br/>老舍创作**《猫城记》**，被视为中国第一部具有现代意义的、文学性较强的长篇科幻小说。 |
| **1953年** | **奖项设立**<br/>首届**雨果奖**在第11届世界科幻大会上颁发，成为历史最悠久的科幻奖项之一。 | — |
| **1954年** | — | **新时期起点**<br/>郑文光在《中国少年报》发表**《从地球到火星》**，被视为新中国第一篇有影响力的科幻小说。 |
| **1960s** | **黄金时代与新浪潮**<br/>以阿西莫夫、克拉克等为代表的“黄金时代”科幻达到高峰，随后“新浪潮”运动兴起，转向更文学化、内省的主题。 | — |
| **1978年** | — | **科幻热潮**<br/>全国科学大会召开，“科学的春天”到来。叶永烈**《小灵通漫游未来》**出版并引发轰动；童恩正**《珊瑚岛上的死光》**在《人民文学》发表并获奖。 |
| **1980年** | — | **影视化尝试**<br/>电影**《珊瑚岛上的死光》**上映，被认为是第一部真正意义上的国产科幻类型片。 |
| **1980s** | **赛博朋克兴起**<br/>威廉·吉布森的**《神经漫游者》**(1984)定义了“赛博朋克”流派，深刻影响了未来科技文化。 | **科文之争与沉寂**<br/>科幻文学经历“姓科还是姓文”的争论，随后进入一个相对低谷期。 |
| **1986年** | — | **银河奖创立**<br/>《科幻世界》杂志（前身《科学文艺》）设立**“银河奖”**，后发展为中国历史最悠久的科幻文学奖项。 |
| **1991年** | — | **独立与学科化**<br/>《科学文艺》正式更名为**《科幻世界》**。银河奖确立为年度奖；吴岩在北师大开设科幻文学选修课，标志着理论建设开始。 |
| **2010年** | — | **星云奖创立**<br/>由董仁威、姚海军、吴岩等共同发起创立全球华语科幻“星云奖”。 |
| **2015年** | — | **走向世界**<br/>刘慈欣**《三体》荣获第73届雨果奖**最佳长篇小说奖，实现亚洲零的突破，标志着中国科幻进入世界舞台中央。 |
| **2019年** | — | **电影元年**<br/>电影**《流浪地球》**上映并获巨大成功，开启国产科幻电影热潮，被定义为“中国科幻电影元年”。 |
| **2020年** | — | **产业政策**<br/>国家电影局、中国科协发布**“科幻十条”**，从政策层面大力推动科幻电影产业发展。 |
| **2023年** | — | **世界主场**<br/>**第81届世界科幻大会在成都举办**，这是该大会首次在中国举行，象征着中国科幻与世界深度融合。 |

:::info 📝 编者注
通过对比可以发现，早期中国科幻多作为“科学传播”或“救亡图存”的工具出现（1903, 1978），而2015年《三体》获奖后，中国科幻真正开始以平视的姿态介入全球科幻文化的对话。
:::
